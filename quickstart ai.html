Quick Start

  1. Clone: git clone git@github.com:garet113/pulse-lang.git
  2. Install: python -m pip install -e .
  3. Run: pulse example.pulse

First Program

let temp = sensor("temp")
let hum  = sensor("humidity")
let comfort = smooth((temp * 0.7) + (hum * 0.3), 3)

@rate 2Hz: { print("comfort:", comfort) }

Time Blocks

  • @rate 60Hz — run every frame
  • @every 1s — periodic
  • @after 5s — one-shot
  • @when event(payload) — event-driven
