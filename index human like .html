Pulse — Reactive, Time‑Native Language

Variables in Pulse are alive — they update automatically when dependencies change. Built-in time blocks (@rate, @every, @after, @when) make real-time code natural.

Get Started Affective AI Guide

Key Features

  • Reactive by default
  • Time-native syntax
  • Previous-state operator ~x
  • Events: emit + @when

Install (local dev)

python -m pip install -U pip
python -m pip install -e .
pulse example.pulse

Example

const TWO = 2
let temp = sensor("temp")
let hum  = sensor("humidity")
let comfort = smooth((temp * 0.7) + (hum * 0.3), 3)
@rate 2Hz: { print("comfort:", comfort) }
